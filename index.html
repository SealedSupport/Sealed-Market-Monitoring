How it works

Step 1 — Weekly Upload

Export For Sale (active) and Recently Sold from RP Data each week. Upload both here. The tool remembers all previous data — new uploads add to the record, not replace it.

Step 2 — Auto Cross-Check

Properties in For Sale that appear in the Sold export are automatically flagged as Sold. A listing is only marked Withdrawn if it's missing from both exports.

Step 3 — Price Tracking

First listed price is locked on initial import. If a new upload shows a different price, it's recorded as "Updated Listed Price" with the original preserved. Sold price can be overridden if Not Disclosed.
